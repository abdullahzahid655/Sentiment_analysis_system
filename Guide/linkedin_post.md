🎬 **Just completed my IMDB Sentiment Analysis Project!** 

After weeks of hard work, I'm excited to share my NLP project that analyzes movie review sentiments with **92.1% accuracy**!

🔑 **What I Built:**
• A complete sentiment analysis pipeline trained on 50,000 IMDB movie reviews
• 7-step text preprocessing (cleaning, tokenization, lemmatization, stopwords removal)
• Multiple ML models (Logistic Regression, Naive Bayes, SVM, Random Forest)
• TF-IDF feature extraction for optimal performance
• Production-ready FastAPI REST API for real-time predictions
• Interactive dashboard with visualizations (ROC curves, confusion matrices, word clouds)

📊 **Results:**
• Model Accuracy: 92.1%
• ROC-AUC: 0.97
• Inference time: ~10ms per review

🛠️ **Tech Stack:**
Python | scikit-learn | NLTK | FastAPI | Pandas | Matplotlib

### 2. Capture Screenshots

**Screenshot 2 - Test the /predict Endpoint:**

- Click "Execute"
